Heart Disease Prediction — ML Pipeline نظام التنبؤ بمخاطر أمراض القلب بالذكاء الاصطناعي

Developed a comprehensive Machine Learning pipeline for predicting heart disease risk, trained on the CDC BRFSS dataset containing 319,795 patients. The system employs state-of-the-art predictive modeling and explainability techniques to assist clinical decision-making.

Key Features & Technical Highlights:

  • 6 ML Architectures: Evaluated Logistic Regression, Random Forest, Neural Networks (MLP), XGBoost, LightGBM, and Stacking Ensembles.
  • Optuna Optimization: Performed Bayesian hyperparameter tuning over 200 trials, selecting LightGBM V2 as the best model with ROC-AUC = 0.8404 and Recall = 80.57%.
  • Explainable AI (XAI): Integrated SHAP (Shapley Additive exPlanations) values to provide global importance and individual waterfall predictions.
  • Streamlit Predictor: Deployed an interactive web application that accepts real-time patient parameters and displays dynamic SHAP risk analysis.

تطوير خط معالجة متكامل للتعلم الآلي (Machine Learning Pipeline) للتنبؤ بمخاطر أمراض القلب، تم تدريبه على مجموعة بيانات CDC BRFSS الضخمة التي تشمل 319,795 مريضاً. يقدم النظام نمذجة تنبؤية متقدمة مع إمكانية تفسير القرارات لدعم التشخيص الطبي الطبيعي.

أبرز الميزات والتقنيات المستخدمة:

  • 6 نماذج ذكاء اصطناعي: اختبار وتقييم نماذج الانحدار اللوجستي، الغابات العشوائية، الشبكات العصبية (MLP)، XGBoost، LightGBM، ونموذج التجميع التراكمي (Stacking Ensemble).
  • ضبط الفائقة باستخدام Optuna: تنفيذ ضبط المعلمات عبر خوارزميات الاستدلال البايزي لـ 200 محاولة، ليحقق نموذج LightGBM V2 دقة ROC-AUC = 0.8404 واستدعاء Recall = 80.57%.
  • الذكاء الاصطناعي القابل للتفسير (XAI): دمج قيم SHAP لتوضيح العوامل الأكثر تأثيراً على مخاطر الإصابة لكل مريض على حدة وبشكل عام.
  • تطبيق ويب تفاعلي: بناء تطبيق ويب باستخدام Streamlit يقبل مدخلات المرضى في الوقت الفعلي ويعرض تحليلاً ديناميكياً للقيم والمخاطر.
